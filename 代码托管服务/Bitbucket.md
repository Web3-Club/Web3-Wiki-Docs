# Bitbucket
Bitbucket 是一种基于 Git 的托管和源代码存储库服务，是 Atlassian 对 GitHub、GitLab 等其他产品的替代品。<br><br>
Bitbucket 通过 Bitbucket Cloud（Atlassian 的服务器）、Bitbucket Server（客户的本地）或 Bitbucket Data Center（客户本地或云环境中的服务器数量）提供托管选项。
